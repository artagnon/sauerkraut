# Sauerkraut

A mishmash of various matrix multiplication algorithms, with hard-coded shape
(square) and type (int) of matrices, for simplicity.

## Usage

```sh
$ cd sauerkraut-build
$ cmake -GNinja ..
$ ninja check
```
